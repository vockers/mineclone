# Mineclone - Voxel Game
**A simple Minecraft Clone written in C++ OpenGL and SDL2**

![preview_01.png](assets/images/preview_01.png)

## Build Instructions
### Prerequisites
- **SDL2**
- **GLEW**
- **freetype**

```
git clone https://github.com/vockers/mineclone.git
cd mineclone
make
./mineclone
```
